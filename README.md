1. Clonar repositorio
2. Ejecutar comando para cargar modulos de node 
   ```

   ```
3. Clonar archivo **.env.template** en un archivo **.env** 
4. Cambiar las variables de entorno 
5. Ejecutar comando para levantar base de datos

   docker compose up -d
